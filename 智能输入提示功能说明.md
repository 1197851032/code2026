# 智能输入提示功能使用说明

## 🎯 功能概述

智能输入提示功能为用户提供了一个便捷的购物咨询体验，通过智能提示、快捷问题模板和自动补全等功能，大大减少用户的输入量，提高客服体验。

## ✨ 核心特性

### 1. 智能建议
- **实时匹配**：根据用户输入实时提供相关建议
- **智能排序**：按相关性对建议进行排序
- **上下文感知**：基于输入内容动态调整建议

### 2. 快捷问题模板
- **分类管理**：按商品推荐、价格查询、购买咨询等分类
- **一键选择**：点击即可填充问题
- **展开收起**：支持展开查看更多分类

### 3. 热门问题
- **高频问题**：显示用户最常问的问题
- **快速访问**：一键选择热门问题
- **动态更新**：根据使用频率更新热门列表

## 📱 使用方法

### 基础使用

1. **开始对话**
   ```
   访问智能客服页面
   系统自动创建新会话
   ```

2. **输入问题**
   ```
   在输入框中输入问题
   系统实时显示智能建议
   ```

3. **选择建议**
   ```
   点击建议项自动填充
   或继续输入自定义问题
   ```

### 高级功能

#### 快捷问题使用
```
1. 查看快捷问题分类
2. 点击"展开"查看所有分类
3. 选择感兴趣的分类
4. 点击具体问题模板
5. 系统自动发送问题
```

#### 热门问题使用
```
1. 在输入框为空时查看热门问题
2. 点击热门问题标签
3. 系统自动填充并发送
```

## 🔧 API接口

### 获取所有模板
```http
GET /smart-input/templates
```

### 获取相关提示
```http
GET /smart-input/relevant?input=用户输入
```

### 获取智能建议
```http
GET /smart-input/suggestions?input=用户输入
```

### 获取热门问题
```http
GET /smart-input/popular
```

### 搜索模板
```http
GET /smart-input/search?keyword=关键词
```

## 🎨 界面组件

### SmartInput 组件
```vue
<SmartInput 
  :session-id="sessionId"
  @send="handleSendMessage"
  @response="handleReceiveResponse"
/>
```

#### Props
- `sessionId`: 会话ID（必需）

#### Events
- `send`: 发送消息时触发
- `response`: 接收到回复时触发
- `input`: 输入内容变化时触发

### SmartChat 页面
完整的智能聊天页面，包含：
- 对话历史管理
- 消息显示
- 智能输入组件
- 会话切换功能
